比IFCONFIG更好用的IP命令

By viscript

2月 24 2015 更新日期:2月 24 2015

ip命令比ifconfig命令更加强大

ip是iproute2软件包里面的一个强大的网络配置工具，它能够替代一些传统的网络管理工具，例如ifconfig、route等，使用权限为超级用户。几乎所有的Linux发行版本都支持该命令。

  • ip ad sh
    ad：address sh：show
  • ip ad help
  • ip ad
  • ip ad add 182.168.1.254/255.255.255.0 dev eth0
    再次用ifconfig查看，并没有显示添加的新的IP地址。但是用ip ad sh能看到。因为ifconfig未必能支持它的接口。

ip help

ip还能添加策略路由。

ip link help

IP设置好之后，看网络是否连接：

网线插好，不知道是链接到eth1还是0.

  • mii-tool

  • mii-tool -v 有的网卡是不支持的。

    信息回馈：operation not supported

mii-tool --help

  • mii -tool

若要让其启动生效：

vi /etc/sysconfig/network-scripts/ifcfg-eth0
cd !
cp ifcfg-eth0 ifcfg-eth0:8
vi !$
给这个网卡添加第二个IP
service network restart

system-config-network-tui 文本界面。 -gui图形化界面。

netconfig -d eth1:1不怎么用了

PS：用到IP命令之后的注意事项

ip ad add 的命令，service …restart 不生效了
要将其写到rc.local
