iqn（iSCSI Qualified Name）

格式：

iqn.yyyy-mm.[:identifier]
iqn.2014-12.cn.openlab:storage001
iqn：表示“iSCSI Qualified Name”，简称iqn。
yyyy-mm：表示年份-月份。这里是2014-12。
reversed domain name：倒过来的域名:cn.openlab。
identifier：表示识别代码:storage001。

    1. 配置target端：

      5.1 targetcli进入内部命令模式：
      5.2 设置SCSI服务器
      5.2.1建立block
      /> /backstores/block create storage001 /dev/sdb1
      Created block storage object storage001 using /dev/sdb1.
      5.2.2建立IQN
      /> /iscsi create iqn.2014-12.cn.openlab:storage001
      Created target iqn.2014-12.cn.openlab:storage001.
      Created TPG 1.
      5.2.3建立LUN号
      /> /iscsi/iqn.2014-12.cn.openlab:storage001/tpg1/luns create /backstores/block/storage001
      Created LUN 0.
      5.2.4建立ACL （此处添加客户端IQN）
      /> iscsi/iqn.2014-12.cn.openlab:storage001/tpg1/acls create iqn.2014-12.cn.openlab:client001
      Created Node ACL for iqn.2014-12.cn.openlab:client001
      Created mapped LUN 0.
      5.2.4建立portal (此处为服务器IP)
      />/iscsi/iqn.2014-12.cn.openlab:storage001/tpg1/portals create 192.168.232.134
      Using default IP port 3260
      Created network portal 192.168.232.134:3260.
      5.2.4 查看并退出
      1.ls 查看
      2.exit退出
      Global pref auto_save_on_exit=true
      Last 10 configs saved in /etc/target/backup.
      Configuration saved to /etc/target/saveconfig.json
